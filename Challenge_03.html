Text Editor

Text editors are computer programs used to edit plain text. The following are examples of text editors used by developers;

  • TexPad
  • UltraEdit
  • Dreamweaver
  • Komodo
  • PsPad
  • Vim
  • Notepad
  • Aptana
  • Brackets

Each text editor has its advantages so it best a developer chooses the one he is most comfortable with. Here is a list of my top three text editors;

  1. Visual Studio Code
  2. Sublime Text
  3. Atom
